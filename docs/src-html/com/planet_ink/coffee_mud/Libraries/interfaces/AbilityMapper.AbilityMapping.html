034/**
035 * The Ability Mapper Library handles everything dealing with the relationship
036 * between players, races, etc and Abilities (Skills, Spells, Prayers, etc).
037 * The principle duty is to map the provider of player Abilities to those
038 * Abilities, along with any requirements for being provided the skills,
039 * and the condition the skill is in when it is received.
040 * 
041 * @author Bo Zimmerman
042 *
043 */
