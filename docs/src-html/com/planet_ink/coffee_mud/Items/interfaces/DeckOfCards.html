024/**
025 * An interface for a deck of playing cards, like you'd use to play
026 * Gin or Poker or something. Includes methods for generating a Hand
027 * of cards, shuffling, and so forth.
028 * @see HandOfCards
029 * @author Bo Zimmerman
030 */
031public interface DeckOfCards extends HandOfCards
032{
033        /**
034         * Creates a new deck of 52 cards of the same time as the
035         * underlying object.  The deck is then handed over to
036         * the mob or room specified. 
037         * @param owner a mob or room to receive the new deck.
038         * @return the new deck of 52 cards, not shuffled.
039         */
040        public DeckOfCards createDeck(Environmental owner);
041        
042        /**
043         * Resets the deck back to 52 cards.  It will
044         * grab cards from all external sources first
045         * and return them to the deck container owner.
046         * If this fails to produce 52 cards, it will
047         * create a set of cards for the deck.
048         * this method also destroys any hands being
049         * managed.
050         * @return true 
051         */
052        public boolean resetDeckBackTo52Cards();
053        
054         /**
055         * If a hand of cards has previously been added to this
056         * deck for internal management, this method will return
057         * that hand given the player object.
058          * @param player the player whose hand to look for
059          * @return the hand belonging to that player, or null
060          */
061        public HandOfCards getPlayerHand(MOB player);
062
063        /**
064         * Adds and possibly creates a hand for the given player
065         * if no hand is passed in, a new empty one is created
066         * the hand is then added to our table, keyed by the player
067         * object
068         * @param player the player whose hand this will be
069         * @param cards the hand, or null to make a new one
070         * @return the hand of cards passed in or created
071         */
072        public HandOfCards addPlayerHand(MOB player, HandOfCards cards);
073
074        /**
075         * If the given player object has a hand of cards currently
076         * being managed by this deck, this method will remove all
077         * of the cards from the hand, return them to the deck,
078         * then remove the hand from management, and destroy the hand.
079         * @param player the player whose hand to remove.
080         */
081        public void removePlayerHand(MOB player);
082        
083        /**
084         * This method adds to the base functionality found
085         * in HandOfCards.java by ensuring that all cards
086         * added to the deck are added face down.
087         * @param card the card to add back to the deck.
088         */
089        @Override
090        public boolean addCard(PlayingCard card);
091}
