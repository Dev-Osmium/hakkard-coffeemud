038/**
039 * The core of event handling in CoffeeMud, a CMMsg represents an event that
040 * occurs.  All events are caused by MOBs (even natural events).   CMMsg objects
041 * are usually created by calling the CMClass classloader:
042 * @see com.planet_ink.coffee_mud.core.CMClass#getMsg(MOB, int, String)
043 *
044 * Source
045 * The source of any message must always be a valid reference to an instance of the
046 * MOB interface. In short, all events that occur in the system are a direct result
047 * of the activity of a MOB. This is on the theory that the universe is controlled
048 * and governed by sentience. In the extremely rare instances where a mob is not
049 * readily available to provide a message source, one should be instantiated --
050 * even if it is just a blank, new StdMOB.
051 *
052 * Target
053 * The target of a message may be null, or any valid reference to an instance of the
054 * Environmental interface, which includes Items, MOBs, Rooms, Exits, etc. The type
055 * and context of message you wish to generate will typically tell you intuitively
056 * whether the source is doing something to someone or something else, or is acting
057 * independently. This is usually another mob or an item, but you will find examples
058 * of all kinds of targets in the code.
059 *
060 * Tool
061 * The tool of a message may be null, or any valid reference to an instance of the
062 * Environmental interface, which includes Items, Abilities, MOBs, Rooms, Exits,
063 * etc. The tool represents something which the source is utilizing to accomplish
064 * the task or generate the event. This is typically either an Ability object (like
065 * a Spell or Skill being used), or an Item object (like a weapon in an attack event).
066 *
067 * Source Code
068 * This is an encoded integer which represents what the source MOB is actually doing.
069 * Codes are separable into major and minor codes, with the major code being a bitmask
070 * and the minor code being a constant integer representing the event type.  See below.
071 *
072 * Source Message
073 * This is the string which the source MOB will see should the event occur successfully.
074 *
075 * Target Code
076 * This is an encoded integer which represents what is happening to the target.
077 * If there is no target, this number will typically have the value of 0 (NOEFFECT).
078 * Codes are separable into major and minor codes, with the major code being a bitmask
079 * and the minor code being a constant integer representing the event type.  See below.
080 *
081 * Target Message
082 * This is the string which the target MOB (if it is a MOB) will see should the event
083 * occur successfully. If there is no target, this string is null.
084 *
085 * Others Code
086 * This is an encoded integer which represents how any other objects (such as MOBs,
087 * Items, Rooms, Exits) other than the source and target, in the same room, perceive
088 * the event. If the event is completely imperceptible by anything other than the
089 * source, it may be 0 (NOEFFECT) Codes are separable into major and minor
090 * codes, with the major code being a bitmask and the minor code being a constant
091 * integer representing the event type.  See below.
092 *
093 * Others Message
094 * This is the string which other MOBs in the same room as the source and target
095 * MOBs will see should the event occur successfully. If the event is completely
096 * imperceptible by other MOBs, it may be null.
097 *
098 *
099 * ** * ** * ** * ** * ** * ** * ** * ** * ** * ** * ** * ** * ** * ** * ** * ** * ** * ** * **
100 * You should also familiarize yourself with the interface that handles these
101 * events
102 * @see com.planet_ink.coffee_mud.core.interfaces.MsgListener#okMessage(Environmental, CMMsg)
103 * @see com.planet_ink.coffee_mud.core.interfaces.MsgListener#executeMsg(Environmental, CMMsg)
104 * @see com.planet_ink.coffee_mud.core.interfaces.MsgListener
105 * @see com.planet_ink.coffee_mud.Locales.interfaces.Room#send(MOB, CMMsg)
106 * @see com.planet_ink.coffee_mud.Locales.interfaces.Room#sendOthers(MOB, CMMsg)
