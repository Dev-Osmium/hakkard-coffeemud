038/**
039 * A Faction is an arbitrary numeric range, where different mobs/players can be
040 * within that range, if they have the faction at all. Factions can be
041 * programmatically set to change due to events that occur to/around the mob,
042 * and adjust themselves relative to other factions. Subsets of the faction can
043 * be given readable names for display to the user.
044 *
045 * @see com.planet_ink.coffee_mud.MOBS.interfaces.MOB#fetchFaction(String)
046 * @see com.planet_ink.coffee_mud.MOBS.interfaces.MOB#addFaction(String, int)
047 */
